COMPILAR E RODAR:
Caso não consigo compilar e rodar o programa por métodos alternativos, indicamos o uso do seguinte comando no terminal do diretório onde se encontra os arquivos (GrafoED2).

Compilar Grafo de Matriz: gcc GrafoMatriz.h GrafoMatrizMain.c -o GrafoMatriz.exe
Rodar o programa: ./GrafoMatriz.exe

Compilar o Grafo de Listas: gcc GrafoLista.h GrafoListaMain.c -o GrafoLista.exe
Rodar o programa: ./GrafoLista.exe
